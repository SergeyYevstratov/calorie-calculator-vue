# Курсова: Калькулятор калорій 

Веб-додаток для розрахунку показників тіла та добової потреби в калоріях: **BMR**, **BMI**, **TDEE**, а також **макросів (білки/жири/вуглеводи)** з візуалізацією у вигляді **кругової SVG-діаграми**. Є **вхід/реєстрація** через LocalStorage та **світла/темна тема**.

---

## Що зроблено
- Сучасний інтерфейс у форматі **3 колонок (1–2–3 як сервіс)**
- Світла/темна тема (перемикач у меню)
- Розрахунок **BMR, BMI, TDEE**
- Розрахунок **макросів** + **кругова діаграма (SVG)**
- Логотип **PNG** у хедері
- Вхід/реєстрація (LocalStorage)
- Сторінки: **Пояснення**, **Про проєкт**

---

## Функціонал
- Введення даних: вік, зріст, вага, стать, активність
- Автоматичні розрахунки:
  - **BMI** (індекс маси тіла) + базова інтерпретація
  - **BMR** (базовий обмін)
  - **TDEE** (добова норма калорій з урахуванням активності)
- Макроси (Б/Ж/В) у грамах та відсотках
- Збереження користувача та сесії в LocalStorage

---

## Сторінки
- Головна (калькулятор)
- Вхід /Реєстраці
- Пояснення (що таке BMR/BMI/TDEE та як користуватися)
- Про проєкт
